## SPK-MapresPNJ

Projek Web Sistem Pendukung Keputusan (SPK) untuk membantu admin atau panitia mapres dalam menyeleksi Mahasiswa Berprestasi yang ada di lingkungan kampus

### Tools Used

1. HTML
2. PHP
3. CSS
4. MySQL Database
5. JavaScript

### Feature

1. Login
2. Add Mahasiswa
3. Add Criteria
4. Preview Score
