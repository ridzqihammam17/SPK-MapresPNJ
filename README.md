## SPK-MapresPNJ

Mini Projek Web untuk Mata Kuliah Sistem Pendukung Keputusan (SPK) berfungsi untuk membantu admin dan panitia mapres dalam menyeleksi Mahasiswa Berprestasi yang ada di lingkungan kampus

### Tools Used

1. HTML
2. PHP
3. CSS
4. MySQL Database
5. JavaScript

### Feature

1. Login
2. Add Mahasiswa
3. Add Criteria
4. Preview Score
